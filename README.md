# BalmJS template - `simple`

> A simple project

## Requirements

```sh
npm i -g balm-core
npm i -g balm-cli
```

```sh
# Verify
balm -V

# Output:
# balm-cli: 2.0.0
# balm-core: 3.0.0
```

## Installation

```sh
balm init simple my-project

cd my-project
yarn
# or
npm i
```

## Enjoy

```sh
# For development
npm run dev

# For production
npm run prod
```
